# Lexical Rules

| Term / Lexeme     | Pattern              |
| :---------------- | :------------------: |
| identifiers       | [a-zA-Z][a-zA-Z0-9]* |
| integer constants | [0-9]+               |
| left parenthesis  | (                    |
| right parenthesis | )                    |
| plus              | +                    |
| times             | *                    |
